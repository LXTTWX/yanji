家檐安记

📖 项目简介

家檐安记是一款同时覆盖年轻人健身自我管理与长辈健康远程守护的纯本地、零后端、跨平台家庭健康工具。

它用 Flutter 一套代码覆盖 App / Web / 桌面三端，用 Hive 把健康数据锁在本地，用 AES 加密和加盐哈希守护隐私安全，用场景化视觉与轻量情感设计，把冰冷的健康记录变成有温度的家庭纽带——一边帮年轻人坚持自律，一边替子女默默守护家人。

🎯 解决的核心痛点

痛点 说明
🏋️ 健身难坚持 独居健身缺少正向反馈与进度追踪，惰性难以克服
💔 关怀难开口 频繁询问长辈健康双方尴尬，疏于询问又内心牵挂
🔒 隐私难保障 主流健康App强制注册联网，中老年人对数据泄露高度敏感
📱 工具难联动 健身与健康管理工具割裂，缺少代际数据联动与情感连接
📰 信息难辨别 健康资讯一搜全是广告，长辈难以筛选权威信息

🏃 健身模式 · 自我成长管理

功能 说明
身体基础档案 身高/体重/BMI/BMR自动计算，30天体重趋势图
健身计划打卡 3套7天入门方案（减脂/增肌/肩颈放松），一键打卡
数据可视化 连续打卡天数、周/月完成率、累计训练天数
成就勋章系统 连续打卡7天/30天、体重达标解锁勋章，生成分享卡片
激励语陪伴 打卡后随机弹出鼓励型/轻松型激励语

👨‍👩‍👧‍👦 守护模式 · 长辈健康关怀

功能 说明
多成员档案 独立存储健康数据，备注病史/用药/过敏史
健康指标记录 血压/心率/步数/服药打卡，异常自动标红+通俗解释
趋势分析 周/月血压心率变化图，支持导出CSV供就诊使用
适老化设计 大字体/大按钮/高对比度，一步直达核心功能
首页概览 所有家人最新健康状态，异常置顶提醒

🔐 隐私与安全

功能 说明
应用启动锁 4位数字密码 + SHA-256加盐哈希存储（不存明文）
防暴力破解 连续错误5次锁定60秒
敏感数据模糊 首页/健康档案数值默认模糊，点击才显示
自动锁定 切后台超过设定时间自动锁定（1/3/5分钟）
数据全本地 100%存储于本地设备，不上传任何云端

📦 数据流转

层级 方案 说明
L1 本地离线 Hive 数据库 日常单人使用
L2 加密导入导出 AES加密JSON 异地备份与迁移

🚀 快速开始

环境要求

· Flutter 3.22.x 稳定版
· Dart 3.4.x

安装运行

```bash
# 克隆项目
git clone https://github.com/yourusername/jiayananji.git
cd jiayananji

# 安装依赖
flutter pub get

# 生成Hive适配器
flutter pub run build_runner build --delete-conflicting-outputs

# 运行Web端
flutter run -d chrome

# 构建Web端
flutter build web --release --web-renderer canvaskit

# 构建Android
flutter build apk --release

# 构建Windows
flutter build windows --release
```
